Selected Software Projects

As a student, I aimed to develop a broad skillset in order to become a versatile software engineer.
This portfolio highlights the wide array of topics I have learned about and languages I have worked with, including Python, Java, JavaScript, HTML, SQL, and C++.

  • Translator Tool

    Python
    As part of the AISES National Conference miniHackathon, my team built the first iteration of what went on to be Yugtun. The goal was to create a web app to do translations between Yup’ik, a Native Alaskan language, and English.

  • ML Classifier

    Python
    Built a claffifier to predict the presence or absense of heart arrhythmia in patients based on EKG results, and classify the type of arrythmia when they were present. We implented Logistic Regression, Support Vector Machines, and Neural Network models and achieved F1 score of 0.82 on the SVM.

  • Web Security

    JavaScript, Docker
    A 2-part project to introduce OWASP top 10 web vulnerabilities by having us first complete a series of hacks to exploit vulnerabilities on a web application and then implement defenses for these web attacks.

  • Chatbot

    Python
    An interactive recommender system that suggests movies to the user based on the their preferences and sentiment. It incorporates algorithms like Naive Bayes, Porter Stemmer, regular expressions, spell check, and logarithmic based recommendations.

  • Skull (Reinforcement Learning)

    Julia
    This project used Reinforcement Learning Algorithms to train an agent on many iterations of the card game Skull, a game of strategy and calculated guess, and then be able to make optimal decisions while playing.

  • Map Reducer

    C++
    A systems architecture project to build a fully operational MapReduce Framework utilizing multiprocessing, networking, threading, and distributed computing techniques.

  • Web Photo App

    ReactJS, Node.js, MongoDB
    A full stack project to create a single page web application, which mimicked a photo sharing app. We created new components, implemented dynamic view switching, and implemented a server to communicate with a database to fetch data and hold sessions.

  • Dora Web App

    Angular, Node.JS, HTML, CSS
    Dora was a dynamic web application integrated with Yelp and Google Places API's, which allows users to enter a location and a variety of preferences to receive selected suggestions for different meals or activities. This was built for my senior project.

  • Encrypted Chat

    C++
    Worked with another student to implement a secure and efficient end-to-end encrypted chat client using the Double Ratchet Algorithm

  • Bunny World

    Java, SQLite, Android Studio
    A team project to build an android app for a graphical adventure game, that allows for playing the game and editing the game components. The game editor is a complete, graphical, document oriented, OOP/GUI project.

  • This Site

    JavaScript, HTML
    I used a template and then added my own features to spice it up in order to have a space to show off some of the things that I am most proud of. It is a continuous work in progress.

Cybersecurity and Software Engineering Intern | Summer 2018

I spent 12 weeks working for PwC’s advisory in their cybersecurity practice. My client was a large tech company and I worked on their Bug Bounty program, focusing mostly on web vulnerabilities. I also developed an automation tool in JavaScript to help improve the efficiency of assigning reported vulnerabilities to their correct owner.
My work was well-received and at the end of the summer I was evaluated at the highest-level, along with an offer for full-time employment.

  • Bug Bounty

    Working on Bug Bounty I gained incredible exposure to real-life security vulnerabilities. The work focused mainly on web vulnerabilities like XSS and subdomain takeovers. I was involved in the process from beginning to end- triaging vulnerabilities reported on HackerOne, assigning fixers or helping write defenses myself, and validating the solutions of over a dozen bugs.

  • Automation Tool

    Assigning tickets for fixing was often a tedious process that required checking multiple sources before an assignee could be determined. As a solution, I built a dynamic web tool to automate parts of this search process. Using Google Apps Script and Python, I created a tool to perform API calls, parse the information, and map the data to locate the owner of domains. While it was just a prototype, the feedback on the tool was positive and plans were made to build it out further.

Spending a summer with PwC was an incredible experience because of their dedication to developing their people, including their interns. Over the summer we were given opportunities to network with leaders throughout the firm, receive mentoring from people at all levels of their career, and participate in a nation wide case competition,, which my team made to the semi-final round. At the completion of the internship, I received an offer for full-time employment which included a week-long trip to Disney World with everyone else who received an offer. This trip was jam packed with networking, team-building, and professional development opportunities.

About me

Hi there! Thanks for stopping by my page. My name is Kathryn Bunner and I recently graduated from Stanford University with a B.S. in Computer Science, focusing on information and data.

I also received a minor in Native American Studies, which offered a great balance to my technical coursework, enhanced my ability to effectively articulate and defend complex topics, and increased my perspective on the many issues that impact contemporary Natives and other marginalized groups.

Beyond school, I served as President of the American Indian Science and Engineering Society, Financial Officer of the 2016 Stanford Powwow (one of the largest powwow’s in the country), and the Academic Chair for the Tridelta Omega Chapter. I also spent time tutoring undergraduates in computer science and a local middle school girl in learning to code.

Currently, I am a Software Engineering Resident at Google in Mountain View, CA.
