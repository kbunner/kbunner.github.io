Software Projects

Below you will find descriptions of some of my favorite projects. I have included GitHub links where I can. My preferred language is Python, and I have a fair amount of experience with Java, JavaScript, HTML, C++ and SQL. I have also worked with Julia, C, Android, and Google Apps Scipts before.

  • Yugtun

    Python
    As part of the AISES National Conference miniHackathon, my team built the first iteration of what went on to be Yugtun. It is a web app to do translations between Yup’ik, a Native Alaskan language, and English.

  • Dora

    JavaScript, HTML, CSS, Node.JS
    Dora was a dynamic web application built for my Senior Project. Dora allows users to enter a location and a variety of preferences and receive three suggestions for different meals or activities.

  • Skull

    Julia
    This project used Reinforcement Learning Algorithms to train an agent to make optimal decisions in the card game, Skull.

  • Chatbot

    Python
    This project is an interactive Chatbot on the terminal that recommends movies based on the user’s preferences and sentiment. To do this it incorporated algorithms like Naive Bayes, Porter Stemmer, regular expressions, spell check, and logarithmic based recommendations.

  • Map Reducer

    C++
    A systems architecture project to build a fully operational MapReduce Framework utilizing multiprocessing, networking, threading, and distributed computing techniques.

    JavaScript, Docker
